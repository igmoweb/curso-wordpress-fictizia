Plugins de ayuda al desarrollo

Query monitor

Plugin multifunción para revisar cada detalle de la ejecución.

https://wordpress.org/plugins/query-monitor/

Emails Log

Permite crear un log para guardar todos los emails que sen envían desde la instalación

https://wordpress.org/plugins/email-log/

Theme Check

Permite revisar si un Tema cumple los estándares de WordPress

https://wordpress.org/plugins/theme-check/

Developer plugin

Metaplugin que permite instalar otros plugins útiles para desarrollar.

https://wordpress.org/plugins/developer/
